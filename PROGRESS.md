## ✅ Phase 1: 프로젝트 뼈대

### 비개발자 설명
> 집을 짓기 전에 땅을 고르고 기초 공사를 한 것과 같습니다.
> 아직 방 안에 가구는 없지만, 방의 위치와 구조는 정해진 상태입니다.

### 뭐가 만들어졌나?
- **9개 페이지 틀** - 메뉴 클릭하면 각 페이지로 이동 (내용은 비어있음)
- **헤더/푸터** - 모든 페이지 상단 메뉴, 하단 연락처
- **한/영 전환 버튼** - 언어 변경 시스템 준비
- **데이터 파일** - 제품, 설비, 회사정보 담을 파일들

### 확인 방법
```
http://localhost:5174 접속 → 메뉴 클릭해보기
```

## ✅ Phase 2: 견적문의 기능

### 비개발자 설명
> 고객이 온라인으로 견적을 요청하면, 담당자 이메일로 자동 전송되는 시스템입니다.
> 전화/팩스 대신 웹사이트에서 바로 문의할 수 있습니다.

### 뭐가 만들어졌나?

**고객이 보는 화면:**
```
┌─────────────────────────────────────┐
│         견적문의 양식                │
├─────────────────────────────────────┤
│ 회사명 *      [입력란]              │
│ 담당자명 *    [입력란]              │
│ 이메일 *      [입력란]              │
│ 연락처 *      [입력란]              │
│ 제품종류 *    [볼트/너트/기어/...]   │
│ 예상수량      [입력란]              │
│ 문의내용 *    [입력란]              │
│ 파일첨부      [도면 등 첨부 가능]    │
│                                     │
│        [ 견적 요청하기 ]            │
└─────────────────────────────────────┘
```

**흐름:**
```
고객 작성 → 버튼 클릭 → 담당자 이메일 수신 → 고객에게 "접수완료" 표시
```

### 확인 방법
```
http://localhost:5174/quote 접속
```

### 이메일 발송 설정 (아직 안 함)
실제 이메일을 받으려면 `.env` 파일에 Gmail 설정 필요:
```
SMTP_USER=회사이메일@gmail.com
SMTP_PASS=Gmail앱비밀번호
QUOTE_RECIPIENT=견적받을이메일
```

## ✅ Phase 3: 제품 소개

### 비개발자 설명
> 카탈로그처럼 제품들을 보여주는 페이지입니다.
> 볼트/너트/기어 카테고리별로 필터링할 수 있습니다.

### 완료된 것
- 제품 목록 페이지 (`/products`)
- 제품 카드 (사진 + 이름 + 규격)
- 카테고리 필터 (전체 / 볼트 / 너트 / 기어)

---

## ✅ Phase 4: 회사소개 + 기타 페이지들

### 비개발자 설명
> 회사 정보, 설비 현황, 인증서 등을 보여주는 페이지들입니다.

### 완료된 페이지들
| 페이지 | 경로 | 내용 |
|--------|------|------|
| 회사소개 | `/about` | CEO 인사말, 연혁, 인증서, 오시는 길 |
| 설비/기술 | `/equipment` | 생산설비, 검사장비 |
| R&D | `/rnd` | 핵심기술, 특허, 인증 |
| ESG | `/esg` | 환경경영, 안전보건, 윤리경영 |
| 품질관리 | `/quality` | 품질방침, 프로세스, 인증 |
| 채용안내 | `/careers` | 복리후생, 채용공고 |

## ✅ Phase 5: 메인 페이지

### 비개발자 설명
> 웹사이트 첫 화면입니다. 회사의 첫인상을 결정합니다.

### 완료된 것
- Hero 배너 (슬로건 + 견적문의 버튼)
- 신뢰 지표 (업력 30년, 직원 30명, 월 100만개, 인증 3개)
- 주력 제품 3종 하이라이트
- 핵심 설비 섹션
- 하단 CTA 배너 (견적문의 유도)
